HTML

ウェブページの構造をマークアップします。SEO,アクセシビリティを意識したコーディングを心がけています。

css

webページのレイアウトを作成します。またサイト運用まで視野に入れた保守性の高いコードを行います。

javascript

Webページにアニメーションなどの動きを付けます。また検索や拡大表示といった機能を作成します。

Illustrator

シンプルなロゴやアイコン、図解などの制作を行います。線や図形を活かしたクリーンなデザインが得意です。

Photoshop

画像の明るさの調整、バナーづくりなどの画像編集を対応できます。見た目にもわかりやすいサイト作りをサポートします。

お知らせやブログを自分で更新できるWebサイトを制作できます。「簡単に使える」「わかりやすい」を大事にしています。
